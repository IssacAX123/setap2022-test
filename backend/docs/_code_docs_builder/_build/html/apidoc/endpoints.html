class endpoints.task.Task(*args, **kwargs)

Bases: flask_restx.resource.Resource

Task Class representing all endpoints related to tasks

Task class has the following methods: GET / .get() to get get task details POST / .post() to add a new task PATCH / .patch() to update an existing task DELETE / .delete() to delete a task
